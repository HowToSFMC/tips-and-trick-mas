# Simple Static Site for GitHub Pages

This is a basic static site template with HTML, CSS, and JavaScript, ready to deploy to GitHub Pages.

## Deployment Instructions
1. Commit your changes to the `main` or `master` branch.
2. Go to your repository settings on GitHub.
3. Under "Pages", set the source to the root (`/`) of the `main` or `master` branch.
4. Save and your site will be live at `https://<your-username>.github.io/<your-repo>/`.

## Files
- `index.html`: Main page
- `style.css`: Stylesheet
- `script.js`: JavaScript

Feel free to customize this template!
